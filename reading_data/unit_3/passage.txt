新マーク発表

A市の青木市長は、チューリップの花をモチーフにした新しい市のマークを発表した。
A市市役所の[広報課]{こうほうか} ( 19-a ) 、この新しいマークは青木市長からのメッセージを ( 19-b ) 。マークのまんなかに、A市の花であるチューリップをデザインし、まわりの小さい[模様]{もよう}は市民が楽しく生活している様子を表しているという。このマークは、区長が親しみやすさと、あたたかさを ( 20 ) ものにしたいと考えてもとになるアイデアを出し、A市に住むデザイナーに頼んで作った。
A市は、今日から市のホームページ ( 21 ) 、この新しいマークを公開する。また、近日中にこのマークをもとにしてキャラクターを作り、その名前を市民から広く[募集]{ぼしゅう}する予定だ。
日本では、各市町村にいろいろなキャラクターがいて、人気者になっている。とても ( 22 ) は、ハンカチなどの商品になったり、テレビで取り上げられることもある。A市のキャラクターにも、たくさんの人に愛される名前を考えて ( 23 ) と、市長は語っている。
